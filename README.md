# Telegram bot style transfer
------  
The final project of the deep learning school from MIPT (1 sem, fall 2022)
------  
Инициализация бота:  
```bash  
echo API_TOKEN=<YOUR_TELEGRAM:TOKEN> > .env
```
------
Запуск бота:
```bash  
docker-compose up --build
```
